
# Walmart Sales Data Analysis  
### End-to-End SQL + Python Project (MySQL Focused)

**Real-World Business Intelligence Project** | Data Cleaning • Feature Engineering • MySQL • Complex SQL Analytics

---

## Project Overview
This is a **complete end-to-end data analysis project** on **Walmart Sales Dataset (10K+ transactions)** that combines:
- **Python (Pandas)** → Data cleaning, transformation & automation
- **MySQL** → Advanced analytics using complex SQL queries
- **Kaggle API** → Automated dataset download

The goal is to answer **15 critical business problems** across **Sales, Revenue, Customer Behavior, Profitability, Branch Performance & Operational Efficiency**.

---

## Project Pipeline
**Kaggle API → Data Cleaning → Feature Engineering → Load to MySQL → 15 Business Problem Solving via SQL → Insights & Documentation**

15 Walmart Business Problems Solved (With SQL)
Category A — Sales & Revenue Analysis

#,Business Question,Purpose
Q1,"Analyze Payment Methods and Sales

Q2,"Total Quantity Sold by Payment Method
How many items were sold through each payment method?",Understand sales distribution across payment channels
Q3,"Best-Selling Category per Branch
Which product category sells the most in each branch?",Optimize branch-level product placement and inventory
Q4,"Revenue Contribution by Category
What percentage of total revenue does each category contribute?",Identify major revenue-driving categories
Q5,"Average Basket Size
What is the average number of items purchased per invoice?",Understand customer buying behavior and cross-sell effectiveness
Q6,"Highest Revenue Day per Month
Which day in each month generated the highest revenue?",Identify peak revenue days and seasonal patterns
Q7,"Total Profit by Category
What is the total profit generated by each category?",Identify high-profit categories
